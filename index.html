Personal Project

Personal Project

"Zero Waste" 를 주제로 하나의 쇼핑몰 페이지를 완성하는 개인 프로젝트입니다.

프로젝트 개요

“AboutPlastic” 은 제로웨이스트를 주제로 한 개인 프로젝트 쇼핑몰 웹사이트입니다.
가상의 브랜드를 설정하고, 다양한 쇼핑몰 사이트의 레이아웃을 벤치마킹하여 하나의 완성도 있는 웹사이트로 구현했습니다.
브랜드는 폐플라스틱을 재활용해 새로운 오브제나 제품을 만들어낸다는 콘셉트에서 출발했으며, 슬로건은 “Let’s Talk About Plastic !” 으로 정했습니다.
민트색과 갈색을 포인트 컬러로 사용해 친환경적이고 정적인 브랜드 이미지를 강조했으며, 전체적으로 감각적인 분위기를 연출하고자 했습니다.

2025 Art Craft Week

Team Project

Team Project

흩어진 행사의 정보를 한곳에 모으고, 고유한 브랜드 이미지를 강화하기 위해 "2025 공예주간" 공식 홈페이지를 디자인·구현한 팀 프로젝트입니다.

프로젝트 개요

「2025 공예주간」은 대한민국 공예문화의 저변 확대와 산업 활성화를 목표로 한 전국 단위 공공 축제입니다. 하지만 기존에는 SNS와 일부 기관 사이트에만 정보가 파편적으로 제공되어 참여 접근성과 행사 정체성 확보에 한계가 있었습니다. 이에 우리는 공예주간만의 고유한 브랜딩과 정보 허브 역할을 수행할 공식 웹사이트를 기획·제작했습니다.


팀원들과의 포지셔닝 과정을 통해 ‘약간은 정적이고 예술적’인 방향을 전체적인 디자인 콘셉트로 설정했습니다. 또한 공식 포스터의 색감이 이 방향성과 잘 어울린다고 판단하여, 포스터의 컬러 팔레트를 웹사이트 전반의 시각적 아이덴티티로 적용했습니다.


나의 역할

저는 메인 페이지의 거점도시프로그램 섹션과 서브 페이지 중 거점도시프로그램, 로그인, 회원가입 페이지 제작을 담당했습니다. 디자인 방향에 맞춘 레이아웃과 색감 적용을 통해, 사용자 친화적인 인터페이스와 통일성 있는 브랜딩을 구현하는 데 기여했습니다.

2025 IPRF

Team Project

Team Project

「2025 인천 펜타포트 락 페스티벌」의 행사 특성과 정보 전달력을 강화하기 위해 공식 웹사이트를 리뉴얼한 팀 프로젝트입니다.

프로젝트 개요

「2025 인천 펜타포트 락 페스티벌」은 한국을 대표하는 락 페스티벌 중 하나이지만, 기존 공식 웹사이트는 행사 분위기와 달리 다소 정적인 톤을 띠고 있었습니다. 또한 프로그램 관련 정보가 각 서브페이지에 충분히 정리되어 있지 않아 사용자가 원하는 내용을 찾기 어렵다는 한계가 있었습니다.


이에 따라 페스티벌의 에너지와 현장감을 시각적으로 반영하고, 동시에 정보 구조를 개선하여 사용자 친화적인 경험을 제공하는 방향으로 웹사이트 리뉴얼을 진행했습니다. 색감, 레이아웃, 인터랙션을 통해 젊고 다채로운 브랜드 이미지를 강화하고, 주요 프로그램과 관람 정보를 한눈에 확인할 수 있도록 체계화했습니다.


나의 역할

저는 이 프로젝트에서 로고 제작을 비롯해 메인 페이지의 서머리 섹션, 서브 페이지 중 관람 안내 가이드와 프로그램 소개 페이지를 담당했습니다. 디자인 방향에 맞는 시각적 요소와 사용자 흐름을 구현함으로써, 전체 사이트의 완성도를 높이는 데 기여했습니다.

Personal Project
